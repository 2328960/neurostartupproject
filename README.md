# NeuroStartUp
NeuroStartUp - динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта.

## Начало работы
Для получения копии этого проекта для запуска на локальном ПК нажмите на кнопку Download.

### Prerequisites
Для работы на ПК потребуются:
* Git
* Любой браузер

### Установка и запуск
Откройте файл .exe и следуйте инструкции установщика
